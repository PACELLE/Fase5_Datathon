# Datathon - Pós-Tech FIAP (Fase 5)

## 📌 Sobre o Projeto
Este repositório contém a solução desenvolvida para o **Datathon da Fase 5** da Pós-Tech em Data Analytics (FIAP). O objetivo principal do projeto é aplicar técnicas avançadas de análise de dados, machine learning e storytelling para resolver um desafio de negócio real proposto durante o encerramento do curso.

## 🚀 Objetivo
Analisar grandes volumes de dados para extrair insights acionáveis e desenvolver um modelo preditivo/prescritivo que auxilie na tomada de decisão estratégica, focando em performance e escalabilidade.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Python 3.x
* **Análise de Dados:** Pandas, NumPy
* **Visualização:** Matplotlib, Seaborn ou Plotly
* **Machine Learning:** Scikit-learn (ou bibliotecas similares como XGBoost/LightGBM)
* **Ferramentas:** Jupyter Notebook / Google Colab

## 📂 Estrutura do Repositório
* `notebooks/`: Contém os arquivos Jupyter com a limpeza, exploração e modelagem dos dados.
* `data/`: (Opcional) Base de dados utilizada ou instruções para download.
* `reports/`: Documentação complementar ou apresentações de resultados.
* `src/`: Scripts Python auxiliares (caso existam).

## 📊 Principais Etapas
1.  **Exploração de Dados (EDA):** Identificação de padrões, outliers e correlações.
2.  **Engenharia de Features:** Tratamento de variáveis e criação de novos atributos.
3.  **Modelagem:** Treinamento e validação de algoritmos de Machine Learning.
4.  **Avaliação:** Análise das métricas de performance (Acurácia, RMSE, F1-Score, etc.).
